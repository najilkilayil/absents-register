# Absent Register Record

Absent Register record is a easy and friendly useable website for teachers to take attendece and mark absentees easily and in timeless.

This website is completely made using HTML,CSS, and JS.

In this website the user can input the date and the student name, the input datas will save in local storage when the user clicks the save button and it will shows the table with printing each datas. 

And it also shows the totale number of the absentees of that day.
